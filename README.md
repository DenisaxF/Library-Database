# Library Database Management System

A Java-based library management system designed to efficiently handle books, authors, categories, and user interactions, with persistent storage using a MySQL database.

This project was developed as part of an Object-Oriented Programming course and focuses on clean architecture, database interaction, and real-world use cases.

# Features

- Manage different types of items (books, comics, etc.)

- Author and category management

- CRUD operations for library entities

- MySQL database integration using JDBC

- Audit system that logs actions into a CSV file

- Custom exceptions for error handling (e.g. duplicate items, item not found)

- Modular and object-oriented design

# Technologies Used

- Java

- MySQL

- JDBC

- CSV file handling

- IntelliJ IDEA
